Eclipse plugin

Enhancement using the Eclipse plugin

Eclipse APT

Eclipse setup for Query bean generation (via Java annotation processor).

The Eclipse IDE plugin is installed from: http://ebean.io/eclipse/update

Recommendation

Use both IDE plugin enhancement (Idea or Eclipse) plus build time enhancement (maven or gradle).

That is, the IDE plugin is good to use during development enhancing the beans as the IDE compiles the classes. The IDE plugin can perform the enhancement during development and unit testing. For production you can then choose between build time enhancement (via maven or gradle) or run time enhancement via javaagent.

Query bean generation with Maven and APT

We can use APT to generate Query beans. To do this with Eclipse and maven have a look at Query bean generation with Eclipse + APT + Maven.

Installing the plugin for Eclipse

Help > Install new software

http://ebean.io/eclipse/update

Install Ebean 11.x eclipse plugin

Turn on plugin per project

Project > Enable Ebean enhancer

The plugin is turned on and off per project. We need to turn it on for it to perform enhancement on the current project.

Review Messages

Window > Show view > Other > Error log

When we are compiling entity beans we can go to the Error log view to confirm that the plugin is enhancing the entity beans.
