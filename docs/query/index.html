Type safe queries (Query beans)

We can generate "Query Beans" which provide the ability to write type safe queries. 

// Example using Customer query bean
List<Customer> customers =
  new QCustomer()
      .name.startsWith("Rob")
      .status.equalTo(Customer.Status.NEW)
      .findList();
