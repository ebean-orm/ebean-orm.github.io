068 * // explicitly register the entity beans to avoid classpath scanning
069 * config.addClass(Customer.class);

071 *
072 * Database database = DatabaseFactory.create(config);
073 *
074 * }</pre>
075 *
076 * <p>
077 * Note that ServerConfigProvider provides a standard Java ServiceLoader mechanism that can
078 * be used to apply configuration to the ServerConfig.
